# Pour Tail

Pour Tail is a premium destination for exceptional cocktails and bar experiences. This project is a modern web application built to showcase the art of mixology with a stunning, interactive user interface.

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/) (GreenSock Animation Platform)

## Features

- **Interactive UI:** engaging animations and transitions using GSAP.
- **Responsive Design:** Optimized for various screen sizes using Tailwind CSS.
- **Modern Typography:** Custom fonts for a unique brand identity.
- **Component-Based Architecture:** Modular and reusable React components.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Contains the main application code (pages, layout, globals.css).
- `components/`: Reusable UI components (Hero, Cocktails, Menu, etc.).
- `public/`: Static assets like images and fonts.
